#  4.13 上传下载

finalshell可以可视化操作

如果看不到目录，使用root用户在finalshell中登录
# rz，sz命令
需要安装软件包 lrzsz
### 下载
```shell
sz 路径
```
### 上传
```shell
rz
```
回撤执行后会弹出文件选择框，让你选择
