#  3.4 修改权限控制

## chmod命令（修改权限）（文件所属用户/root用户才可用）
```shell
chmod [-R] 权限 文件或文件夹
```
- -R：递归修改
PS：权限有两种书写格式，数字格式/字母格式
### 权限表示（数字格式）：
```shell
chmod [-R] xxx 文件或文件夹
```
数字模式的三位分别表示：**所属用户、所属组、其他用户**的权限
每位的数字是对应权限之和（权限数字如下）
| 权限 | 数字 |
| --- | --- |
| r（读） | 4 |
| w（写） | 2 |
| x（执行） | 1 |
### 权限表示（字母格式）：
```shell
chmod [-R] u=rwx g=rwx o=rwx 文件或文件夹
```
其中：**u**表示所属用户（user），**g**表示所属组（group），**o**表示其他用户（others），**a**表示所有人（all，相当于ugo）
可以使用`+`、`-`、`=`来添加、删除、设置权限，例如：
- `chmod u+x,g-w,o=r 文件名`
- `chmod a+x 文件名`（给所有人添加执行权限，等同于 `chmod u+x,g+x,o+x 文件名`）
## chown命令（修改所属用户/组）（仅root权限）
```shell
chown [-R] [用户][:][用户组] 文件或文件夹
```
选项说明：
- -R：递归修改
- 冒号：用于分隔用户和用户组
PS：
只改用户组，需要带冒号
只改用户不用带冒号
示例：
`chown :group1 test.txt`（只修改所属组为group1）
