#  2.7 文件操作命令

# touch命令（创建文件）

### 语法
```shell
touch 路径
```

touch命令无选项，参数必填，文件名直接写在路径里，可以使用相对/绝对/特殊路径
# cat命令（查看文件内容）（一次全显示）
是concatenate（连接、串联）的简写
cat命令还有将多个文件连接显示的作用
### 语法
```shell
cat 路径
```
Cat没有选项，只有参数（必填）
# more命令（查看文件内容）（分页显示）

### 语法
```shell
more 路径
```
PS：
1. 分页显示过程中，通过空格键翻下一页
1. 点击q（Quit）键退出查看
1. 回车键一行一行显示
# cp命令（复制）
cp：copy的简写
### 语法
```shell
cp [-r] 参数1 参数2
```
PS：
1. -r 选项，表示复制文件夹时递归复制（复制子文件/文件夹）
1. 参数1，被复制的路径
1. 参数2，目标路径
# mv命令（移动/重命名）
mv：move的缩写
### 语法
```shell
mv 参数1 参数2
```
PS：
1. 参数1，要移动的路径
1. 参数2，目标路径
1. **重命名：在目标路径中写入新的文件名即可**
# rm命令（删除）
rm：remove的简写
### 语法
```shell
rm [-r -f] 参数1 参数2 ...
```
PS：
1. -r选项：递归删除（**删文件夹一定要带-r，否则删不了**）（文件夹非空不非空不带r都删不了）
1. -f选项：强制删除（需要root用户）
1. 参数是被删除的路径，可以一次删多个
1. **rm命令支持通配符*，可以模糊匹配**
