#  几个头文件作用

### <REGX52.H>寄存器地址定义文件
将寄存器地址转换成标识符供调用
不同单片机该文件不同
### <INTRINS.H>编译器内部函数声明
**就是快捷指令：如_nop_（空操作指令），堆栈操作，浮点数状态检查**
不同编译器该文件不同
