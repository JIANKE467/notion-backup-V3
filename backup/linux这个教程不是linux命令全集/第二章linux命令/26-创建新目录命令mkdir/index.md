#  2.6 创建新目录命令（mkdir）

### 语法
```shell
mkdir [-p] 路径
```
### 选项
-p：自动创建不存在的父目录（当一次性创建多层时）
### 参数
路径使用绝对路径或相对路径都行

读法：麦克dir
