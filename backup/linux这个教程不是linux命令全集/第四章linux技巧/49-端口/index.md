#  4.9 端口

端口分
物理端口：USB，HDMI
虚拟端口：内部虚拟端口，与外部交互
IP只能确定某台电脑，而端口能锁定计算机内的某程序
# Linux端口划分（并非强制约束）
### 公认端口：1~1023
**为系统和知名软件预留，尽量不要占用**
ssh：22端口
https：443端口
### 注册端口：1024~49151
**可随意使用，固定绑定程序服务**
### 动态端口：49512~65535
**不固定绑定程序，程序临时使用**
# 查看端口占用命令nmap（端口扫描）
需要安装软件包：nmap
查看命令：
```shell
nmap IP地址
```
# 查看指定端口/进程占用端口情况netstat
需要安装软件包：net-tools
```shell
netstat -anp | grep 端口号或进程号
```
