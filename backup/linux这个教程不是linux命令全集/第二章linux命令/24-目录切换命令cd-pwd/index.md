#  2.4 目录切换命令（cd / pwd）

# cd命令（切换目录）
### 语法
```shell
cd [路径]
```
**PS：cd 命令不写路径则回到默认路径（/home/用户名）**

# pwd命令（显示当前目录）
### 语法
```shell
pwd
```
是print working directory（打印当前工作目录）的缩写
无参数，无选项
