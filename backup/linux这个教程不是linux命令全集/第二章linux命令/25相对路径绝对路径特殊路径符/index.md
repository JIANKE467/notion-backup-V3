#  2.5相对路径/绝对路径/特殊路径符

# 相对路径/绝对路径
**绝对路径必须从根目录开始**
**相对路径只需要从当前路径开始往后写**
举个例子：
当前在 “/home” 要去 “/home/ssbn”
用绝对路径：
```shell
cd /home/ssbn
```
用相对路径：
```shell
cd ssbn
```

# 特殊路径符
| 特殊路径符 | 作用 | 例子 |
| --- | --- | --- |
| . | 当前路径 | cd ./ssbn 与
cd ssbn效果一致 |
| .. | 上一级目录 | cd ..  代表返回上一级目录
cd ../..  代表返回上两级目录 |
| ～ | 默认目录（/home/用户名） | cd ~ 代表转到默认目录 |
