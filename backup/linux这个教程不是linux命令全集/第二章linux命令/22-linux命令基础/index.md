#  2.2 Linux命令基础

先做一个区分：
命令行指Linux terminal（终端）
命令指在终端中输入的命令
# 通用格式
```shell
command [-options] [parameter]
```
command：命令
-options：选项
parameter：参数
[]：代表非必填
