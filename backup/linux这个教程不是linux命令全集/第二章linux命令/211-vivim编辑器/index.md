#  2.11 vi/vim编辑器

vi/vim是linux的文本编辑器
vim是vi的升级版，完全兼容vi
### 语法
```shell
vim 路径
```
**如果该文件不存在，则自动创建**
# vi/vim有三种模式
### 命令模式（普通模式）
使用vim会先进入普通模式
普通模式下可以移动光标位置，复制粘贴等快捷键操作
该模式下按特定键会进入编辑模式或命令模式
| 按键 | 功能 |
| --- | --- |
| yy | 复制当前行 |
| p | 粘贴 |
| dd | 删除当前行 |
| u | 撤销上一步操作 |
| gg | 跳到文件首行 |
| G | 跳到文件末行 |
| h/j/k/l | 左/下/上/右移动光标 |
| x | 删除光标所在字符 |
| dw | 删除一个单词 |
| 0 | 移动光标到行首 |
| $ | 移动光标到行尾 |
| /关键字 | 向下搜索关键字 |
| ?关键字 | 向上搜索关键字 |
| n | 切换到下一个搜索结果 |
| N | 切换到上一个搜索结果 |
| Ctrl+f/ pagedown键 | 向下翻页 |
| Ctrl+b/pageup键 | 向上翻页 |
| : | 进入底线命令模式 |
还有很多……
### 输入模式（编辑模式）
**在命令模式下，按A,a,I,i,Oo,R,r 进入编辑模式**
该模式下所有的按键只被当做字符处理
**按esc退出编辑模式**
| 按键 | 功能 |
| --- | --- |
| a | 在光标后插入内容 |
| A | 在当前行末尾插入内容 |
| i | 在光标前插入内容 |
| I | 在当前行开头插入内容 |
| o | 在当前行下方新建一行插入 |
| O | 在当前行上方新建一行插入 |
| r | 替换光标所在字符 |
| R | 进入替换模式，连续替换字符 |
| Esc | 退出编辑模式，返回命令模式 |

### 底线命令模式（命令模式）
该模式可以保存/退出文件
**编辑器无法一直停留在命令模式**
**只能在普通模式下按下特定的按键，进入命令模式执行完相应操作后立刻自动退出到shell或退出到普通模式（取决于执行的命令）**
| 按键 | 功能 |
| --- | --- |
| :wq | 保存并退出 |
| :q | 退出（未修改文件时） |
| :q! | 强制退出，不保存修改 |
| :w | 保存文件 |
| :w! | 强制保存 |
| :set nu | 显示行号 |
| :set nonu | 取消显示行号 |
| :/关键字 | 查找关键字（按n查找下一个） |
| :%s/旧/新/g | 全局替换 |
