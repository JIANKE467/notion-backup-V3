## ASCII码推算
只需记住几个具有代表性字符的十进制ASC11码值，即可算出常用字符的ASCII码了。
1. ‘0’的十进制ASCII码为48
1. ‘A’的十进制ASCII码为65
1. 'a'的十进制ASCII码为97
1. 大小字母ASCI码差32
数字及字母只需将上一个字符的十进制ASCII码加一

### 强制类型转换
强制类型转换的一般形式如下：
```c
（类型名） （表达式）
```
```c
int i = 10;
float f;
f = (float)i;// 强制转换int为float类型
```
