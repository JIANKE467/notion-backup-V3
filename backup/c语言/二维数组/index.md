#  二维数组

## 声明
```c
类型名 数组名 [行大小][列大小];
```
## 引用
```c
数组名 [行下标][列下标];
```
## 初始化
```c
int a [3][3]={1,2,3,4,5,6,7,8,9};
```
将所有数组所有元素排列到一行初始化
