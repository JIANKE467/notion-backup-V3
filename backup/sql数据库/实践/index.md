#  实践

[基础概念](基础概念/index.md)
  ssms：是微软SQL数据库图形化管理器
  注册新服务器：不是创建服务器，而是保存已有服务器到这台电脑上
  注意⚠️：注册/连接本地服务器，服务器名要写localhost或localhost/实例名（需在安装SQL时手动设置，如果没有设置则是localhost）
[第一节](第一节/index.md)
  ### 安装SQL服务器
  1. 直接下载微软SQL服务器并安装即可（选择常规安装则服务器均为默认配置，如果想要在一个机器上部署多个SQL服务器，或要修改SQL服务器名称则需要自定义安装）
  这里安装的是SQL服务器本体
  ### 使用ssms连接/注册SQL服务器
  1. 下载ssms（图形化管理器）
  ps：ssms只是一个管理器不是服务器本体，不能实现添加服务器的功能
  1. 直接连接服务器/注册服务器到电脑上（相当于保存cookie，下次可直接登录）
  ps：连接/注册，分本地服务器和远程服务器，本地服务器在服务器名栏填localhost或localhost/名称（第二种需要自定义安装），远程服务器服务器名直接输入IP
  服务器名的位置实际填的是服务器地址，个人认为翻译不准确
  注册：注册服务器在视图→已注册的服务器→右键相应的分组（没有相应分组自己创建一个）→新建服务器注册
  或：直接在对象资源管理器中点连接→连接完成后右键相应的服务器→注册
  ### 本地服务器的开关
  方法一：Windows服务中找到SQL Server带几个服务右键开或关
  方法二：在电脑中搜索SQL server Configuration manager→SQL server服务→右键SQL server（mssqlserver）开，关，重启
  PS：SQL server bowers服务是解析和监听端口的，使用远程连接需要打开
