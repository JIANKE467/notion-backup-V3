#  指针


## 声明
```c
数据类型*指针变量名;
```
ps：指针变量在内存中的全部空间存的都是别的变量的内存地址（指针是一个地址容器）
## 赋值
赋值为变量地址
```c
指针变量名=&变量名
```
ps：&是取地址符
不能将变量直接赋给指针变量（要先取地址再赋值）
## 初始化
```c
数据类型*指针变量名=&变量名
```
或初始化为空指针
## 调用
调用指针要带*号，
但要调用/修改指针所存的的地址则不带*号
```c
printf("%d",p);
```
其中p是指针变量
