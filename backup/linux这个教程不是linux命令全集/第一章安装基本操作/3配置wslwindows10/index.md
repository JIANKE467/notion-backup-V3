#  3.配置wsl（Windows10+）

### WSL是Windows下的linux子系统
在设置→应用和功能→程序和功能→启用和关闭Windows功能→勾选Linux的Windows子系统
### 然后再微软商店直接搜索Ubuntu下载即可
### 建议一并下载Windows terminal
Windows terminal：更好用的cmd
### 在Windows terminal中使用Ubuntu
打开Windows terminal
点加号旁边的折叠按钮
选择Ubuntu
