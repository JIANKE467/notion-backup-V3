#  1.2.4计算机系统的层次结构

**M0微程序机器**
将二进制指令拆成多个步骤执行的层级
**M1传统机器**
就是正常识别二进制的层级
**M2虚拟机器**
操作系统的层级
**M3虚拟机器**
使用汇编语言的层级
**M4虚拟机器**
使用高级语言的层级
