#  2.3 ls命令（列出目录内容）

# 格式
```shell
ls [-选项] [路径]
```
选项有：
| **选项** | 名称 | 解释 |
| --- | --- | --- |
| -a | all（全部） | 显示隐藏文件 |
| -l | long format
(长格式显示) | 显示详细信息，并且竖向显示 |
| -h
(要与-l一起用，否则没用) | Human reader
(可读性增强) | 显示文件大小单位 |
**不使用选项和参数时，默认以****横向****列出****当前目录****所有内容。**
**PS：Linux终端启动时默认加载目录： “/home/用户名”**
