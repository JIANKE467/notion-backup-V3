#  4.5 日期/时区

# date命令（查询系统时间）
  ```shell
date [-d] [加减参数] [+格式化字符串]
  ```
  - -d：在当前时间上进行加减
  - 不带任何参数选项，会显示当前年，月，日，星期，时，分，秒，时区
  - 格式化字符串可以组合使用，可以与字符混合使用
  ### **加减参数：**
  +/-与
  year 年
  month 月
  day 天
  hour 小时
  minute 分钟
  second 秒
  组合使用
  如：（加1月，减3天）
  ```shell
date -d "+1 month -3 days"
  ```
  **如果一次性同时加减多个时间单位，如同时加减月和天，要用双引号引起来，避免shell在空格处断句**

  ### **格式化字符串：**

  | 格式化字符 | 含义 |
  | --- | --- |
  | %Y | 完整年份（如：2026） |
  | %y | 年份后两位（如：26） |
  | %m | 月份（01-12） |
  | %d | 日期（01-31） |
  | %H | 小时（00-23） |
  | %M | 分钟（00-59） |
  | %S | 秒（00-59） |
  | %s | 时间戳：从1970-01-01 00:00:00 UTC到现在的秒数 |
  例子：
  1:按照2026-01-01 的格式显示
  ```shell
date +%Y-%m-%d
  ```
  2:按照2022-01-01 10:00:00 的格式显示
  ```shell
date "+%Y-%m-%d %H:%M:%S＂
  ```
  PS：这里必须带双引号，因为格式化字符串中有空格，shell会从空格处断开
  **双引号：强制识别为命令和字符的混合（将引号内作为一个整体处理，防止shell从空格处断开）**
  **单引号：强制识别为字符串**
  **反引号：强制识别为命令**
# 修改Linux时区（root权限）（上海时区）
1. 删除 `/etc/localtime`
1. 把 `/usr/share/zoneinfo/Asia/Shanghai `软链接到 `/etc/localtime`
1. 完成✅
# 自动校准时间（ntp命令）
使用ntp程序（需要下载 ntp）
1. 通过yum或apt包管理器安装下载
1. 启动并设置开机自启（systemctr命令）
1. 完成✅（ntp的服务名叫ntpd，启动后会定期联网校准时间）
也可以手动校准
```shell
ntpdate -u ntp服务器
```
