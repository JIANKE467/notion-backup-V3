#  4.1 快捷键/技巧

### ctrl+c 强制停止
- 强制停止
- 命令输入错误，按这个快捷键可跳到下一行（重写）
### Ctrl+d 退出/登出
- 退出账户
- 退出页面（如退出Python）
PS：vi/vim不适用
### history命令（查看命令记录）
### 匹配历史命令前缀并执行（！+前缀）
语法：
```shell
! 历史命令前几个字母
```
从后往前匹配符合的历史命令并执行
### Ctrl+r 搜索历史命令
按下Ctrl+r 并输入相关历史命令包含的字符，会自动匹配到第一个相关命令（从前往后）
回车即可执行，按左右键即可修改
### 光标移动快捷键
Ctrl+a：跳到命令开头
Ctrl+e：跳到命令结尾
Ctrl+键盘左键：向左跳一个单词
Ctrl+键盘右键：向右跳一个单词
### 清屏 Ctrl+l 
等于clear命令
