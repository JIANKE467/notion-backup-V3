# C 基础语法
在 C 语言中，令牌（Token）是程序的基本组成单位，编译器通过对源代码进行词法分析，将代码分解成一个个的令牌。C 语言的令牌主要包括以下几种类型：
1. 关键字（Keywords）
1. 标识符（Identifiers）
1. 常量（Constants）
1. 字符串字面量（String Literals）
1. 运算符（Operators）
1. 分隔符（Separators）
