#  4.4 链接

# 软链接/硬链接是什么？
**软链接就是一个文件的“快捷方式”**，删除软连接不影响源文件，删除源文件，软链接就会失效
**硬链接就是一个文件多个文件名**，要删除该文件需要删除所有的硬链接
# ln命令（创建链接）
```shell
ln [-s] 源路径 新路径
```
- -s：是创建软链接，不带就是创建硬链接
- 文件，文件夹都能链接
