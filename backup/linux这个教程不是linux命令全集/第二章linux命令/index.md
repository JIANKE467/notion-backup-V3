  ```shell
ls [-选项] [路径]
  ```
  选项有：
  | **选项** | 名称 | 解释 |
  | --- | --- | --- |
  | -a | all（全部） | 显示隐藏文件 |
  | -l | long format
(长格式显示) | 显示详细信息，并且竖向显示 |
  | -h
(要与-l一起用，否则没用) | Human reader
(可读性增强) | 显示文件大小单位 |
  **不使用选项和参数时，默认以****横向****列出****当前目录****所有内容。**
  **PS：Linux终端启动时默认加载目录： “/home/用户名”**

[2.5相对路径/绝对路径/特殊路径符](25相对路径绝对路径特殊路径符/index.md)
  # 相对路径/绝对路径
  **绝对路径必须从根目录开始**
  **相对路径只需要从当前路径开始往后写**
  举个例子：
  当前在 “/home” 要去 “/home/ssbn”
  用绝对路径：
  ```shell
cd /home/ssbn
  ```
  用相对路径：
  ```shell
cd ssbn
  ```

  # 特殊路径符
  | 特殊路径符 | 作用 | 例子 |
  | --- | --- | --- |
  | . | 当前路径 | cd ./ssbn 与
cd ssbn效果一致 |
  | .. | 上一级目录 | cd ..  代表返回上一级目录
cd ../..  代表返回上两级目录 |
  | ～ | 默认目录（/home/用户名） | cd ~ 代表转到默认目录 |

  # cp命令（复制）
  cp：copy的简写
  ### 语法
  ```shell
cp [-r] 参数1 参数2
  ```
  PS：
  1. -r 选项，表示复制文件夹时递归复制（复制子文件/文件夹）
  1. 参数1，被复制的路径
  1. 参数2，目标路径
  # mv命令（移动/重命名）
  mv：move的缩写
  ### 语法
  ```shell

  ```
  PS：
  1. 参数1，要移动的路径
  1. 参数2，目标路径
  1. **重命名：在目标路径中写入新的文件名即可**
  # rm命令（删除）
  rm：remove的简写
  ### 语法
  ```shell
rm [-r -f] 参数1 参数2 ...
  ```
  PS：
  1. -r选项：递归删除（**删文件夹一定要带-r，否则删不了**）（文件夹非空不非空不带r都删不了）
  1. -f选项：强制删除（需要root用户）
  1. 参数是被删除的路径，可以一次删多个
  1. **rm命令支持通配符*，可以模糊匹配**

  **在Linux中各种命令都是程序文件**

  ### 语法

  ```shell
which 要查找的命令
  ```
  # find命令（查找文件）
  find命令用来查找文件
  ### 语法（按文件名查找）
  ```shell
find 起始路径 -name "被查找文件名"
  ```
  PS：
  起始路径指定了从哪搜索
  find支持通配符模糊匹配
  ### 语法（按大小查找）
  ```shell
find 起始路径 -size +|-n[kMG]
  ```
  PS：
  - +|-：代表大于或小于
  - n：代表大小
  - kMG：代表单位：k（小写）表示kb，M表示MB，G表示GB
  例子：
  在根目录查找小于10kb的文件
  ```shell
find / -size -10k
  ```
  如果是区间限制（如10k到10M），这么写
  ```shell
find / -size +10k -size -10M
  ```
[2.9 grep，wc和管道符](29-grepwc和管道符/index.md)
  # grep命令（通过关键字过滤出文件的行）
  全名：global regular expression print 全局 正则表达式 打印
  ### 语法
  ```shell
grep [-n] 关键字 文件路径
  ```
  PS：
  - -n：number的意思，表示在结果中显示匹配的行号
  - 关键字：目标行的关键字，当关键字含特殊符号和空格时用“”括起来
  - 文件路径可以作为<u>标准输入</u>（注意⚠️：不能直接在这里写文本内容，因为用户的输入对命令来说不是标准输入，用户的输入是给shell看的，不是给grep命令看的 [ 前面提到过shell≠命令，所有命令都是一个单独的程序 ]）
  # wc命令（文件内容统计）
  全名：word count 数字数（统计）
  但功能可不止数字数
  ### 语法
  ```shell
wc [-c -m -l -w] 文件路径
  ```
  ### 选项
  | 选项 | 说明 |
  | --- | --- |
  | -c | 统计字节数（bytes） |
  | -m | 统计字符数（characters） |
  | -l | 统计行数（lines） |
  | -w | 统计单词数（words） |
  **不加选项默认输出格式：行数 字符数 字节数 文件名**
  # 管道符 “|”
  管道符的作用：将管道符左边命令的输出作为右边命令的输入
  所以要使用grep命令PS的第三条要使用管道符
  例子：将查看test.txt的命令输出作为查找ssbn所在行命令的输入（查找test.txt中ssbn所在行）
  ```shell
cat test.txt | grep ssbn
  ```
[2.10 echo，tail和重定向符](210-echotail和重定向符/index.md)
  # echo命令（在命令行显示指定内容）
  echo命令相当于c语言的 printf() 函数
  ### 语法
  ```shell
echo 要输出的内容
  ```
  PS：建议使用双引号括起要输出的字符串
  **shell中反引号（长这样` `）表示将反引号内视作命令**
  **shell中单引号（长这样 ‘   ‘）表示将单引号内视作字符**
  **shell中双引号（长这样 “  ”）表示双引号内可以写字符和命令的混合**
  # 重定向符
  有两种：（实际上是多种，这里这介绍两种）
  >：将 > 左侧命令的输出覆盖写入右侧的文件
  ```shell
echo "文本" > 文件路径
  ```
  >>：将 >> 左侧的命令输出追加写入右侧的文件
  ```shell
echo "文本" >> 文件路径
  ```
  # tail命令（查看文件尾部）
  tail 尾巴
  查看文件尾部，跟踪文件最新更改
  ### 语法
  ```shell
tail [-f -num] 路径
  ```
  - -f：follow表示持续跟踪
  - -num：表示查看尾部几行，不填默认为10行

  # vi/vim有三种模式
  ### 命令模式（普通模式）
  使用vim会先进入普通模式
  普通模式下可以移动光标位置，复制粘贴等快捷键操作
  该模式下按特定键会进入编辑模式或命令模式
  | 按键 | 功能 |
  | --- | --- |
  | yy | 复制当前行 |
  | p | 粘贴 |
  | dd | 删除当前行 |
  | u | 撤销上一步操作 |
  | gg | 跳到文件首行 |
  | G | 跳到文件末行 |
  | h/j/k/l | 左/下/上/右移动光标 |
  | x | 删除光标所在字符 |
  | dw | 删除一个单词 |
  | 0 | 移动光标到行首 |
  | $ | 移动光标到行尾 |
  | /关键字 | 向下搜索关键字 |
  | ?关键字 | 向上搜索关键字 |
  | n | 切换到下一个搜索结果 |
  | N | 切换到上一个搜索结果 |
  | Ctrl+f/ pagedown键 | 向下翻页 |
  | Ctrl+b/pageup键 | 向上翻页 |
  | : | 进入底线命令模式 |
  还有很多……
  ### 输入模式（编辑模式）
  **在命令模式下，按A,a,I,i,Oo,R,r 进入编辑模式**
  该模式下所有的按键只被当做字符处理
  **按esc退出编辑模式**
  | 按键 | 功能 |
  | --- | --- |
  | a | 在光标后插入内容 |
  | A | 在当前行末尾插入内容 |
  | i | 在光标前插入内容 |
  | I | 在当前行开头插入内容 |
  | o | 在当前行下方新建一行插入 |
  | O | 在当前行上方新建一行插入 |
  | r | 替换光标所在字符 |
  | R | 进入替换模式，连续替换字符 |
  | Esc | 退出编辑模式，返回命令模式 |

  ### 底线命令模式（命令模式）
  该模式可以保存/退出文件
  **编辑器无法一直停留在命令模式**
  **只能在普通模式下按下特定的按键，进入命令模式执行完相应操作后立刻自动退出到shell或退出到普通模式（取决于执行的命令）**
  | 按键 | 功能 |
  | --- | --- |
  | :wq | 保存并退出 |
  | :q | 退出（未修改文件时） |
  | :q! | 强制退出，不保存修改 |
  | :w | 保存文件 |
  | :w! | 强制保存 |
  | :set nu | 显示行号 |
  | :set nonu | 取消显示行号 |
  | :/关键字 | 查找关键字（按n查找下一个） |
  | :%s/旧/新/g | 全局替换 |
