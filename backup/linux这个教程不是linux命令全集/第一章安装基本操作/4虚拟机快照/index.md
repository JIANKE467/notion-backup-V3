#  4.虚拟机快照

Vmware虚拟机支持为虚拟机制作快照
快照可以将当前虚拟机状态保存，在日后出现故障，可以重新恢复到当前状态。
拍快照：VMware左侧栏找到相应虚拟机右键→快照→拍摄快照
恢复快照：…相应虚拟机右键→快照→快照管理器→点击相应快照→点下面“转到”（恢复）
建议关机恢复和拍快照
