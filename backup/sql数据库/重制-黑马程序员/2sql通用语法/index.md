#  2.SQL通用语法

1. SQL语句以分号结尾
1. SQL不区分大小写，关键字建议用大写
1. 单行注释：- -注释内容  或  #注释内容（MySQL特有）
1. 多行注释：/*注释内容*/
1. [ ]是可选项  < >是必选项  { }是可无限重复
