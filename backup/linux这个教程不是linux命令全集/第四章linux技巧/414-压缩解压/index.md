#  4.14 压缩解压

### 压缩包格式：
zip：Linux，win，mac常用
7zip：win
rar：win
tar：Linux，mac常用
gzip：Linux，mac常用
### Linux中常见压缩文件后缀
`.tar`：归档文件，只是简单将文件封装到.tar文件内，压缩率不高
`.gz`：是gzip格式的压缩文件，压缩率较高
# tar/gz包命令：tar（压缩解压）
tar命令对.tar和.gz均有效
**语法：**
```shell
tar [-c -v -x -f -z -C] 参数1 参数2...
```
- -c：创建压缩文件
- -v：显示进度
- -x：解压
- -f：指定压缩包路径（含文件名）
-f选项必须在选项位的最后一个，不使用-f则默认直接输出压缩文件到终端，-f后的第一个路径就是指定路径
- -z：压缩成.gz，不选默认是.tar包
- -C：选择解压路径
（建议单独使用，和其他参数分开，避免歧义）
- 参数1：如果有-f则这个是压缩文件指定路径，否则就是要被压缩的文件
- 参数2~n：要被压缩的文件路径 
ps：
1. .gz解压时要带-z选项，不能只带-x
1. -f放到最后
1. -C建议单独使用↓
```shell
tar -zxvf test.rar -C /home
```
# zip压缩包命令：zip
### 压缩
```shell
zip [-r] 参数1 参数2...
```
- -r：被压缩的包含文件夹
-  参数1：压缩包路径
- 参数2~n：被压缩文件路径
### 解压
```shell
unzip 压缩包路径 [-d] 路径
```
- -d：指定解压路径
PS：解压时有同名内容会被直接替换
