#  2.10 echo，tail和重定向符

# echo命令（在命令行显示指定内容）
echo命令相当于c语言的 printf() 函数
### 语法
```shell
echo 要输出的内容
```
PS：建议使用双引号括起要输出的字符串
**shell中反引号（长这样` `）表示将反引号内视作命令**
**shell中单引号（长这样 ‘   ‘）表示将单引号内视作字符**
**shell中双引号（长这样 “  ”）表示双引号内可以写字符和命令的混合**
# 重定向符
有两种：（实际上是多种，这里这介绍两种）
>：将 > 左侧命令的输出覆盖写入右侧的文件
```shell
echo "文本" > 文件路径
```
>>：将 >> 左侧的命令输出追加写入右侧的文件
```shell
echo "文本" >> 文件路径
```
# tail命令（查看文件尾部）
tail 尾巴
查看文件尾部，跟踪文件最新更改
### 语法
```shell
tail [-f -num] 路径
```
- -f：follow表示持续跟踪
- -num：表示查看尾部几行，不填默认为10行
